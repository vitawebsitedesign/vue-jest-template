# What
This is a project template with the following setup:
* jest
* vue
* babel
* webpack

# Usage
Installation:
```console
git clone https://github.com/vitawebsitedesign/vue-jest-template.git
cd vue-jest-template
npm i
```

Run sample app:
```console
npm run dev
```

Run sample tests:
```console
./node_modules/.bin/jest
```
